# Project Overview

You are building a production-ready End-to-End Data + AI Platform.

Goal:
- Ingest data (batch + streaming)
- Process data using Spark
- Store in PostgreSQL
- Train ML model (churn prediction)
- Serve model via FastAPI
- Build dashboard using Streamlit
- Use Airflow for orchestration
- Use Docker + CI/CD

STRICT REQUIREMENT:
- Code must be modular, production-style
- No monolithic scripts
- Every component must be testable

Definition of Done:
- All services run via docker-compose
- API returns predictions
- Dashboard displays metrics
